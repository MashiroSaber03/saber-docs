# 配置说明

Saber Translator的各个功能离不开调用AI大模型，若您还不熟悉如何使用API调用AI大模型，那么在开启您的机翻之旅前，请务必阅读本文档，了解目前各大主流的AI大模型的配置方法。

## 翻译服务配置

#### 您可以在翻译页面顶部的设置菜单中找到各种翻译模式的模型配置菜单，在您进行漫画翻译之前，请务必确保您使用的翻译模式所对应的配置菜单中已经配置好了AI模型。

![翻译服务配置](/images/translation-%20model.png)

其中 AI 视觉 OCR、[高质量翻译](/use/hq-translation)、[AI 校对](/use/ai-proofreading)功能需要配置**多模态模型**，有关何为多模态模型，可查阅[模型类型说明](./model-types)中的解释。如何获取 API Key 请参考[如何获取 API Key](./get-api-key)。

软件能够将您配置过的所有参数保存在本地进行持久化存储，因此建议您在运行软件的第一时间就配置好所有的模型。

## 分析服务配置

#### 您可以在分析页面顶部的设置菜单中找到各种分析模式的模型配置菜单，在您进行漫画分析之前，请务必确保您使用的分析模式所对应的配置菜单中已经配置好了AI模型。

![分析服务配置](/images/analysis-model.png)

软件能够将您配置过的所有参数保存在本地进行持久化存储，因此建议您在运行软件的第一时间就配置好所有的模型。
