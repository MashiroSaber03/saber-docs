# 模型参数说明

本页将向您说明在配置模型时能够进行设置的各项参数。所有参数均已有默认值，您可直接使用默认值进行翻译与分析。若您想自定义参数以获得更好的效果，可阅读此页面了解各个参数的具体含义。

---

## Temperature（温度）

控制模型输出的随机性和创造性。

- **范围**：0.0 - 2.0
- **默认值**：0.3
- **推荐值**：0.3 - 0.5（翻译场景）
- **说明**：
  - **较低值（0.3）**：输出更稳定、一致，适合翻译
  - **较高值（1.0）**：输出更有创造性、多样化，适合创作
  - 漫画翻译建议使用 0.3-0.5，确保译文准确性

---

## RPM（每分钟请求数限制）

控制向 API 发送请求的速率，避免触发服务商的频率限制。

- **默认值**：0（即无限制）
- **推荐值**：根据您的 API 套餐调整
- **说明**：
  - 不同的 API 服务商有不同的 RPM 限制
  - 设置过高会导致请求被拒绝，设置过低会影响翻译速度
  - 建议根据您的 API 套餐文档设置为限制值的 80%

---

## 批次数量

控制高质量翻译与AI校对时每批发送的图片数量。

- **默认值**：3
- **推荐值**：3 - 5

---

## 强制 JSON 输出

强制模型以 JSON 格式返回结果，提高解析成功率。

- **默认值**：开启
- **推荐值**：保持开启
- **说明**：
  - 开启后，模型会严格按照 JSON 格式输出翻译结果
  - 可以有效避免模型输出格式错误导致的解析失败
  - 部分旧版模型、某些第三方服务不支持该参数
  - 如果遇到报错，可以尝试关闭此选项

---

## 流式调用

控制是否使用流式输出（逐字返回）模式。

- **默认值**：关闭
- **推荐值**：根据需求选择
- **说明**：
  - 可以有效缓解部分情况下API调用超时的问题
  - 部分旧版模型、某些第三方服务不支持该参数
  - 如果遇到报错，可以尝试关闭此选项
