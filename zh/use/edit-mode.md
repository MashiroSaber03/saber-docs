# 编辑模式

编辑模式是 Saber Translator 的核心功能之一，提供对翻译结果的精细控制。您可以逐个气泡调整文本、样式和背景修复效果，支持手动标注、笔刷修复、实时预览等强大功能。

::: tip 特点
- 双图对比预览（原图 + 译图）
- 逐气泡精细编辑
- 修复/还原笔刷工具
- 实时渲染预览
- 丰富的快捷键支持
:::

## 进入编辑模式

使用[普通翻译](./normal-translation)、[高质量翻译](./hq-translation)或[AI校对](./ai-proofreading)完成翻译后，点击图片预览区上方的"切换编辑模式"按钮进入编辑界面。

---

## 界面布局

![](/images/edit-display.png)

编辑模式界面分为三个主要区域：

### 左侧 - 原图预览

显示漫画原图，原文内容保持不变：
- 气泡框高亮显示（绿色边框）
- 点击可选中对应气泡
- 支持缩放和拖拽查看

### 中央 - 翻译图预览

显示翻译后的图片：
- 实时预览翻译效果
- 选中的气泡会高亮显示
- 与左侧原图同步缩放和滚动

### 右侧 - 属性编辑面板

选中气泡后显示的编辑面板，包含：
- 原文显示区（OCR 结果）
- 译文编辑区
- 样式设置（字体、字号、颜色等）
- 操作按钮

## 顶部工具栏

![image-20251220134322892](/images/edit-mode-top-toolbar.png)

### 导航控制

- **图片导航**：切换查看不同图片（显示当前图片序号）
- **气泡导航**：切换选中不同气泡（显示当前气泡序号）

| 按钮 | 功能 |
|------|------|
| **检测** | 对当前图片重新进行文字检测 |
| **批量检测** | 对所有图片重新进行文字检测 |
| **翻译** | 使用当前设置重新翻译选中气泡 |
| **添加** | 进入绘制模式，手动添加新气泡 |
| **删除** | 删除选中的气泡 |

## 右侧编辑面板

![image-20251220134505941](/images/edit-panel.png)

### 文本编辑

**原文（OCR 结果）**

- 显示识别的原文内容
- 提供复制按钮
- 提供 50 音键盘辅助输入日语

**译文**

- 编辑翻译后的文本
- 支持多行文本
- 修改后点击"应用"生效

### 样式设置

| 设置项 | 说明 |
|--------|------|
| **字体** | 选择字体，支持自定义字体 |
| **字号** | 手动输入或使用 A+/A- 快速调整 |
| **排版方向** | 竖排（↓）/ 横排（→）/ 自动（根据检测）|
| **文字消除** | 选择背景修复方式（LAMA 修复 / 纯色填充）|
| **文字颜色** | 设置文字主体颜色 |
| **描边** | 开关描边效果，设置描边颜色和宽度 |
| **旋转角度** | 文字旋转角度（支持 -180° 到 180°）|
| **位置偏移** | X/Y 方向微调文字位置 |

::: tip 自动排版
选择"自动"排版时，系统会根据文字检测结果自动判断每个气泡的排版方向（横排/竖排）。适合混合排版的漫画。
:::

### 操作按钮

| 按钮 | 功能 |
|------|------|
| **应用** | 应用当前气泡的修改并重新渲染 |
| **应用全部** | 将当前气泡的样式应用到所有气泡 |
| **重置** | 重置当前气泡到初始状态 |

## 气泡管理

![image-20251220134848846](/images/text-box.png)

### 添加气泡

1. 点击工具栏"添加"按钮
2. 在图片上拖拽绘制气泡区域
3. 释放鼠标完成添加
4. 新气泡会自动选中，可立即编辑

### 删除气泡

1. 选中要删除的气泡
2. 点击"删除"按钮或按 `Delete` 键
3. 删除后会自动还原该区域的原图背景

### 调整气泡框

选中气泡后可以：
- **移动**：左键拖拽气泡框内部移动位置
- **调整大小**：左键拖拽气泡框四角或边缘调整大小
- **多选**：按住 `Shift` + 左键点击可多选气泡
- **中键绘制**：按住鼠标中键拖拽可直接绘制新气泡框

### 笔刷工具

编辑模式提供强大的笔刷工具，用于手动修复背景或还原原图：

#### 修复笔刷

**使用方法**：
1. 按住 `R` 键不放
2. 在图片上拖拽涂抹需要修复的区域
3. 释放 `R` 键退出笔刷模式

**功能**：
- 使用 LAMA 模型智能修复涂抹区域
- 适合修复文字残留或背景瑕疵
- 支持调整笔刷大小（滚轮）

#### 还原笔刷

**使用方法**：
1. 按住 `U` 键不放
2. 在图片上拖拽涂抹需要还原的区域
3. 释放 `U` 键退出笔刷模式

**功能**：
- 将涂抹区域还原为原图
- 适合撤销过度修复或恢复背景细节
- 支持调整笔刷大小（滚轮）

**笔刷控制**：
- **调整大小**：笔刷模式下滚动鼠标滚轮
- **笔刷大小显示**：屏幕上会实时显示当前笔刷大小
- **快速切换**：在修复笔刷和还原笔刷之间快速切换

::: tip 使用技巧
- 笔刷大小根据需要调整，细节区域用小笔刷，大面积用大笔刷
- 修复笔刷需要 LAMA 模型支持，确保已选择 LAMA 修复方式
- 还原笔刷可以精确恢复原图的任意区域
:::

## 视图控制

### 布局切换

编辑模式支持两种布局：

| 布局 | 说明 |
|------|------|
| **左右布局** | 原图和译图左右并排显示 |
| **上下布局** | 原图和译图上下排列显示 |

点击工具栏的布局切换按钮可快速切换。

### 视图模式

| 模式 | 说明 |
|------|------|
| **双图模式** | 同时显示原图和译图 |
| **单图模式** | 只显示译图，节省屏幕空间 |

### 缩放控制

- **放大/缩小**：点击 `+` / `-` 按钮或使用鼠标滚轮
- **适应屏幕**：点击 `⛶` 按钮或双击图片
- **原始大小**：点击 `1:1` 按钮
- **同步缩放**：点击 `🔗` 按钮启用，原图和译图同步缩放和拖动

## 快捷键

### 导航快捷键

| 快捷键 | 功能 |
|--------|------|
| `A` / `D` | 切换上/下一张图片 |
| `PageUp` / `PageDown` | 切换上/下一张图片 |

| `U`（按住）+ 左键拖拽 | 还原笔刷模式 |

| 快捷键 | 功能 |

## 使用技巧

### 编辑流程建议

1. 检查气泡检测是否正确，删除误检、添加漏检
2. 检查 OCR 识别结果，必要时手动修正
3. 检查翻译结果，修改不满意的译文
4. 调整样式（字号、颜色、排版方向）
5. 对修复不理想的气泡使用"修复"按钮重新处理
6. 保存会话

### 常见问题处理

| 问题 | 解决方法 |
|------|----------|
| 气泡检测不全 | 点击"添加"手动绘制气泡 |
| 文字显示不全 | 减小字号或调整排版方向 |
| 背景修复有瑕疵 | 点击"修复"重新修复，或使用笔刷手动修复 |
| 文字位置不对 | 使用位置偏移微调，或直接拖拽气泡框 |
| 文字颜色不清晰 | 开启描边效果 |
