# 前言与致谢

## 1. 免费与开源承诺

- **最重要的一点：本工具完全免费且开源，未来也绝不会收费。** 制作这个翻译工具的初衷，只是因为有些漫画可能永远等不到汉化，所以决定自己动手。Saber-Translator 的诞生离不开 GitHub 上众多开发者的无私开源，因此我也会坚守开源精神，将这份便利分享给所有人。
- ⚠️ **请注意：** 我收到了反馈，某些论坛或平台将本工具上架并收费。请大家不要上当受骗，给骗子充钱不如请作者喝杯奶茶（不是）。所有官方发布和更新都在 [GitHub](https://github.com/MashiroSaber03/Saber-Translator) 和 [B站](https://space.bilibili.com/229769907) 进行。

## 2. 感谢每一位支持者

- **非常非常感谢所有赞助过的朋友们！** 训练模型、开发新功能和维护项目需要巨大的时间成本。本人还是个大学生（走过路过的老板给个工作机会好不好 😭😭），大家的支持是我将项目维护下去的最大动力。如果方便，希望大家在赞助时可以备注上自己的网名，后期我会在 GitHub 项目首页专门开辟一个板块来感谢每一位支持者。

## 3. 关于我与项目维护

- 本人目前还是一名即将毕业的学生（26届应届本科生），正在努力寻找工作（**走过路过的老板给个工作机会好不好** 😭😭），因此投入到项目维护上的精力有限。但请放心，大家在使用过程中如果遇到任何 Bug，都可以在 [GitHub Issues](https://github.com/MashiroSaber03/Saber-Translator/issues)、[B站](https://space.bilibili.com/229769907) 或 QQ 群中向我反馈，我会在看到后尽快修复。同样，任何关于功能改进的建议，我都非常欢迎！Saber-Translator 的许多重要更新都源自于用户的宝贵建议。

## 4. 分享与共建

- 如果你觉得这款工具还不错，希望能多多分享给有需要的朋友。更多的用户意味着能更及时地发现潜在问题，也能汇集更多奇思妙想，让 Saber-Translator 变得更好。

## 5. 机翻与尊重

- 尽管 AI 技术日新月异，但机器翻译与专业汉化组的成果仍有差距。如果你计划公开发布使用本工具翻译的作品，**请务必标注为"机器翻译"**。请尊重每一位汉化工作者的辛勤劳动，不要用机翻作品去挑战或贬低汉化大佬们的成果。
